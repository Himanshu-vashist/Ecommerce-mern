Ad Blocker Issue Detected

Your browser's ad blocker is preventing the application from making necessary API requests to our server. This is causing errors like:

Failed to load resource: net::ERR_BLOCKED_BY_CLIENT

Why is this happening?

Ad blockers sometimes block requests to certain API endpoints that match patterns commonly used by tracking or advertising scripts. In our case, some of our legitimate API endpoints are being incorrectly blocked.

How to Fix This Issue

Option 1: Disable Ad Blocker for This Site (Recommended)

The simplest solution is to disable your ad blocker for this website:

  1. Look for your ad blocker icon in your browser's toolbar
  2. Click on it and select "Disable on this site" or similar option
  3. Refresh the page

Common ad blockers:

  • uBlock Origin
  • AdBlock Plus
  • AdGuard
  • Brave Browser's built-in shields

Option 2: Use a Different Browser

If you prefer not to disable your ad blocker, you can try using a different browser without ad blocking extensions installed.

For Developers

If you're the developer of this application, you might want to consider:

  • Renaming API endpoints to avoid common patterns that trigger ad blockers
  • Using a different domain for your API calls
  • Adding this information to your application's documentation
